Документація про Ethernet-кабелі

Цей документ містить огляд основних типів, характеристик та стандартів Ethernet-кабелів, які використовуються в сучасних комп’ютерних мережах.

1. Що таке Ethernet-кабель?

Ethernet-кабель — це фізичний засіб передачі даних у локальних комп’ютерних мережах (LAN). Більшість сучасних Ethernet-кабелів базуються на технології “витаної пари” (Twisted Pair), де дроти скручені між собою для зменшення електромагнітних перешкод.

2. Класифікація за типом екранування

Залежно від способу захисту від перешкод, кабелі поділяються на:

  • UTP (Unshielded Twisted Pair) — неекранована вита пара. Найпопулярніший і найдешевший тип. Не має додаткового захисту, тому чутливий до зовнішніх перешкод.
  • FTP (Foiled Twisted Pair) — вита пара з одним загальним екраном (фольгою) під оболонкою.
  • STP (Shielded Twisted Pair) — вита пара з індивідуальним екрануванням кожної пари. Найкраще захищена від перешкод.
  • SFTP (Screened Foiled Twisted Pair) — поєднання обох методів: кожна пара має фольгу, а весь кабель має загальний екран (сітку або фольгу).

3. Категорії кабелів (Categories)

Категорія кабелю визначає його пропускну здатність та частотний діапазон.

Категорія Максимальна швидкість Максимальна частота Примітки
Cat 5 100 Mbps 100 MHz Застаріла, майже не використовується
Cat 5e 1 Gbps 100 MHz Найпоширеніший стандарт для домашніх мереж
Cat 6 1-10 Gbps 250 MHz Краще захищена від перешкод, ніж 5e
Cat 6a 10 Gbps 500 MHz Оптимальна для дата-центрів та професійних мереж
Cat 7 10 Gbps 600 MHz Високий рівень захисту, вимагає спеціальних конекторів
Cat 8 25-40 Gbps 2000 MHz Оптимальна для коротких дистанцій у серверних кімнатах

4. Розпіновка (Стандарти T568A та T568B)

Для обтиску кабелю використовуються два стандарти кольорового маркування: T568A та T568B. У сучасних мережах найчастіше використовується T568B.

Порядок кольорів для T568B:

  1. Біло-помаранчевий
  2. Помаранчевий
  3. Біло-зелений
  4. Синій
  5. Біло-синій
  6. Зелений
  7. Біло-коричневий
  8. Коричневий

5. Типи підключення

  • Straight-through (Прямий кабель): Обидва кінці обтиснуті за одним стандартом (наприклад, обидва T568B). Використовується для з’єднання різних пристроїв (наприклад, комп’ютер -> комутатор).
  • Crossover (Перехресний кабель): Один кінець T568A, інший — T568B. Використовувався раніше для з’єднання однакових пристроїв (наприклад, комп’ютер -> комп’ютер). Сучасне обладнання має функцію Auto-MDIX, тому цей тип кабелю майже не потрібен.

6. Поради щодо вибору

  1. Для дому: Cat 5e або Cat 6 буде цілком достатньо.
  2. Для геймінгу та стримінгу: Cat 6 забезпечить стабільнішу роботу з меншими затримками.
  3. Для майбутнього (Future-proofing): Якщо ви робите ремонт і прокладаєте кабель у стіни, краще відразу використовувати Cat 6a.
  4. Відстань: Пам’ятайте, що максимальна довжина Ethernet-кабелю без втрати якості становить 100 метрів.
