# Перевірка роботи (модеми)

Так як модеми працюють по аналоговим лініям, для перевірки аналогових ліній потрібні спеціальні аналогові тестери ліній чи сигналів або осцилоскопи. Звичайним абонентам такі інструменти недоступні.

Досить часто, модеми показують в налаштуваннях **рівень сигналу** та **співвідношення сигнал/шум**, які можна використовувати для аналізу сили сигналу та зашумленості лінії. Часто ці показники також обʼєднують у один показник «якість сигналу», або просто кілька поділок.
